Introduction

ValidateThis is a validation framework for ColdFusion objects and structures

Installation : Create a mapping for ValidateThis directory 

<cfset ValidateThisConfig = {definitionPath="/model/"} />
<cfset application.ValidateThis = createObject("component","ValidateThis.ValidateThis").init(ValidateThisConfig) />

Instantiate the ValidateThis.cfc facade object with configStruct

definitionPath points to the folder into which you placed your Rules Definition File
